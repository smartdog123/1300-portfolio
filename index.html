Responsive Redesign

Course Project | CSCI 1300

Solving Problem: Identify usability problems of the original webpage, and practice the workflow of redesigning the webpage

Target Audience: Customers and the owner of the Chongqing House restaurant

Design Iterations: I started by identifying several usability problems, and created low-fi and high-fi prototypes which solved the problems. Lastly I developed the webpage. The main changes include removing duplicated components and unnecessary instructions to improve learnability and memorability, adding responsive design and rearranging the layout to improve visual hierarchy

Conclusion: This project helps me practice some usability principles and how to apply them to the real business

Iterative Design

Course Project | CSCI 1300

Solving Problem: Brainstorm and design an interactive interface for an emerging startup - Bananas

Target Audience: Owner of the Bananas App

Design Iterations: We started by brainstorm low-fi designs for the app and combine ideas into a wireframe. Then we created interactive high-fi mockup and received critiques from peers, and made final iteration which incorporated the feedback. Lastly, we launched user testing, analyzed the results, and contacted the startup to share our mockup

Conclusion: This project helps me go through the full process of iterative design and understand the importance of user testing. I have also learned the importance of clear instruction when conducting user testing
